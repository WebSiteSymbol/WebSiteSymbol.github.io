Инструкция по установке

1

Установите Minecraft Forge

Скачайте и установите Minecraft Forge для версии 1.12.2 с официального сайта

forge-1.12.2-14.23.5.2854-installer.jar
2

Установите Java 8

Если у вас не установлена Java 8, скачайте её с официального сайта Oracle или Adoptium

java -version (должно показывать 1.8.x)
3

Скопируйте мод в папку mods

После установки Forge, откройте папку .minecraft/mods и поместите туда файл клиента

%appdata%/.minecraft/mods (Windows)
~/Library/Application Support/minecraft/mods (Mac)
4

Запустите игру

Запустите Minecraft через Forge профиль и наслаждайтесь!

При открытии меню модов проверьте наличие Nursultan Client
5

Настройка чита

В игре нажмите RShift для открытия GUI и настройте функции под себя

Важно!

Используйте чит только на серверах, где это разрешено. Администрация не несет ответственности за бан вашего аккаунта.
