Инструкция по установке

1

Установите Forge

Скачайте Minecraft Forge 1.12.2 с официального сайта

forge-1.12.2-14.23.5.2854-installer.jar
2

Установите Java 8

Java 8 обязательна для работы клиента

java -version (1.8.x)
3

Скопируйте файл

Поместите ZeusClient.jar в папку .minecraft/mods

%appdata%/.minecraft/mods
4

Запустите игру

Выберите профиль Forge и наслаждайтесь!

Нажмите RShift для меню
