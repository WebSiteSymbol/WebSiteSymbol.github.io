Инструкция по установке

1

Установите Forge

Скачайте Minecraft Forge 1.12.2

forge-1.12.2-xxx.jar
2

Установите Java 8

Java 8 обязательна для работы

java -version 1.8.x
3

Скопируйте файл

Поместите ZeusClient.jar в mods

%appdata%/.minecraft/mods
4

Запустите игру

Выберите профиль Forge

Нажмите RShift для меню
